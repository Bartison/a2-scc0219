# 🧪 Poções e Soluções - Web Service

## Descrição
Web Service completo para a loja "Poções e Soluções", permitindo:
- **Página do comprador**: Visualização de todas as poções com imagens, descrições e preços
- **Painel administrativo**: Cadastro, listagem e remoção de poções
- **API RESTful**: Endpoints para integração com outras aplicações

## Tecnologias
- **Backend**: Node.js + Express
- **Banco de Dados**: SQLite (em memória) com Sequelize ORM
- **Frontend**: EJS + CSS3
- **Comunicação**: JavaScript + AJAX (Fetch API)
